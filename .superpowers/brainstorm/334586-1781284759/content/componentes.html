Lógica de Cálculo (lib/calculations.ts)

Funções puras, sem efeitos colaterais — fácil de testar isoladamente:

  • calcPaintableArea(walls, doors, windows) → m² descontado
  • calcAreaForPaint(paintableArea, coats) → m² × demãos
  • calcPackages(areaForPaint, yieldM2) → Math.ceil()
  • calcTotals(products, extraItems, laborValue) → totais

Server Actions (lib/budget.ts)

  • createBudget(data) → cria rascunho, retorna id
  • updateBudget(id, data) → atualiza qualquer etapa
  • updateBudgetStatus(id, status) → muda status
  • deleteBudget(id) → soft delete

Todas validadas com Zod antes de chamar Prisma.
