Rotas da Aplicação

src/app/
▸ (auth)/
login/      → Tela de login
register/   → Cadastro do pintor
▸ (painter)/
dashboard/  → Painel com lista de orçamentos
orcamento/novo/  → Criar orçamento (multi-step)
orcamento/[id]/  → Ver/editar orçamento
▸ resumo/[id]/
→ Página pública de resumo (compartilhar com cliente)
▸ admin/
page.tsx    → Dashboard admin
produtos/   → CRUD de tintas e rendimentos
pintores/   → Lista de pintores cadastrados
▸ api/
pdf/[id]/   → Gera e retorna PDF do orçamento
auth/[...nextauth]/  → NextAuth handler

Proteção de Rotas

• (auth)/ — público, redireciona pra dashboard se já logado

• (painter)/ — requer sessão NextAuth, qualquer role

• admin/ — requer role === "ADMIN"

• resumo/[id]/ — público (link compartilhável)

• Middleware Next.js cuida de tudo isso em middleware.ts
